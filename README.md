# Flux

> Deep focus tool for AI-assisted developers

## Description

Flux is a CLI tool that helps developers maintain focus by blocking distractions (Discord, YouTube) and providing periodic check-ins. Designed for AI-assisted development workflows, it creates a distraction-free environment so you can stay in the zone.

## Features

- Block distracting websites and applications (Discord, YouTube, etc.)
- Periodic check-ins to verify you're still on task
- Focus session management with customizable durations
- Status tracking to monitor your focus time
- Lightweight and fast (built with Rust)
- Cross-platform support (Linux, macOS, Windows)


## Usage

### Start a focus session

```bash
flux start
```

### Check current status

```bash
flux status
```

### Block specific distractions

```bash
flux block discord youtube
```

### End a focus session

```bash
flux stop
```
